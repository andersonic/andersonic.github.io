Projects

PokéBot
This project used Python to create a competitive Pokémon-playing agent. I implemented the expectimax algorithm, which peaked at ELO 1309 and had a GXE (expected long-term winrate) of 54.4% after 50 battles. I was also responsible for interfacing with the battle simulator using Selenium.
Tower Defense
A tower defense game written with Java, and my final project for AP Computer Science. The player builds defensive units to defend the player's tower and stop an invading force.
DiscoveryInsights
At the Brown Datathon, I worked with a dataset of layperson ratings of academic papers. I investigated potential bias in the ratings based on the author’s perceived gender using SciPy. Happily, I found no significant difference between the ratings of papers by authors with masculine, feminine and ambiguous names.
Froodies
This webapp, built with Flask, connects would-be wasted food with people in need. I worked almost exclusively on the back end, handling food submissions and location-based requests for food.
CheckIn
CheckIn allows the user to set automated daily check-ins with aging friends or relatives, and alerts the user if the check-in goes unanswered. CheckIn uses Python and Swift, and incorporates CMU Sphinx speech-to-text, Amazon Alexa text-to-speech and the DropBox SDK.
