Tower Defense

A tower defense game, and my final project for AP Computer Science. The player builds defensive units to defend the player's tower and stop an invading force. The project uses Java for the logic, as well as the JFrame library for the GUI.

Shoutouts to Alice for being an awesome partner for this project.

Froodies

WhackFall2017 project. This webapp, built with Flask, HTML, and CSS, connects would-be wasted food with people in need. I worked almost exclusively on the back end, handling food submissions and location-based requests for food.

Shoutouts to my collaborators and fellow Wellesley women Marinn, Rachel, and Sandra.

CheckIn

CheckIn allows the user to set automated daily check-ins with aging friends or relatives, and alerts the user if the check-in goes unanswered. CheckIn uses Python and Swift, and incorporates CMU Sphinx speech-to-text, Amazon Alexa text-to-speech, the DropBox SDK, and the JSON library for Python. This was my team's project for HackUMassV.

Shoutouts to my teammates David, Declan and Eric.

Web Search Parser

Uses Python and Selenium to make a Google search, parse the results, and return them in an array. While the returned array is definitely hard for a human to dissect, if you're a computer, it's much easier to use than the results page. The result of this program could be piped into programs that analyze the echo chamber effect, search engine marketing, and more.

AD Info Guide

This is a (somewhat outdated) beginner's guide to the game AD2460, a free-to-play online MMO. I built this primarily to make a modular, relatively readable, and comprehensive guide for new players in my alliance. This was also my first foray into front-end web development.

My thanks to my allies donkeyrider, Legend, pipper, and Zelos for their feedback on the design and content, and to Zelos for the alliance image.

Application Manager

A webapp to manage applications for scholarships and internships. It uses PHP and MySQL to crowdsource data and help users find scholarships and internships that will be a good fit for them. This was my learning project for databases, and is still a work in progress. I'd like to finish implementing deletion and editing of entries in the search table.
